# Content System OS — Banca Empresas

Este repositorio contiene todas las reglas, estándares y guías de UX Writing para Banca Empresas.

## 🎯 Objetivo
Unificar cómo escribimos en todos los productos digitales del banco para garantizar:

- Consistencia
- Claridad
- Escalabilidad
- Uso con IA 
## 🧠 Cómo usar este sistema

### Para UX Writers
Consulta las reglas antes de escribir microcopy.

### Para Designers
Usa estas reglas para definir textos en Figma y prototipos.

### Para PMs
Valida que los flujos respeten el lenguaje del sistema.

### Para IA 
Usa estos archivos como base de:
- generación de texto
- auditoría de UX writing
- validación de flujos

---

## 📦 Estructura

01-voz-y-tono → cómo hablamos  
02-Puntuación en microcopy → cómo escribimos UI  
03-Guías gramaticales → reglas lingüísticas  
04-flujos-y-ui → comportamiento en interfaces  
05-centro-de-ayuda → contenido educativo  
06-Emojis → chat, email, sms  
07-Modales → cómo crearlos correctamente 
08-Nombres de productos → referencia práctica  
09-Uso de mayúsculas

---

## 🚀 Principio del sistema

> “El contenido no se escribe desde cero, se genera y valida desde un sistema.”
